## Java代码
### Hello.java
```
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}
```
### 解释上面的代码：
```
Java程序是大小写敏感的 对格式没有严格要求;
public class 是来定义1个对象的必须小写;
Hello 是对象的名字，首字母必须大写;
花括号必须成对匹配，表示一个逻辑代码块;
缩进不是必须的，但用空格或Tab键缩进会比较好看,能够很容易看出代码块的开始和结束;
缩进是4个空格或者是1个Tab键;
缩进两层是8个空格或者是2个Tab键;
public static void main 是java程序的入口方法，不能修改；
main 后面的小括号包含了方法参数
String 后的 [] 表示程序可以接受字符串数组作为参数
参数的名字通常命名为args 是 arguments 的缩写
System.out.println 表示在屏幕上显示括号内输出的内容并且换行
"Hello, world!" 我们括号内的是字符串，必须用双引号括起来""
因为System.out.println("Hello, world!"); 这一行是JAVA语句，所以结尾需要用英文分好;结尾
最后保存文件的时候，名必须是Hello.java，文件名也要注意大小写，必须与程序定义的类名完全一致，扩展名是.java。
```
### 编译（windows 的记事本可进行编译）
```
文件名必须是 Hello.java
保存类型是：所有文件（*.*）
编码（E）：ANSI
资源管理器中可查看到刚才保存的 Hello.java 文件

命令提示符CMD 进去；cd 切换到 Hello.java所在目录；
如：cd Documents
Documents>dir   可查看到 Hello.java 是否存在

然后编译 javac ，输入 javac Hello.java
无任何输出表示成功，可查看编译出的Hello.class文件。

执行 java Hello
就会输出 Hello, world!

如果编译书写错误，执行 java Hello，会自动报错位置，修改保存重新编译即可

可以改掉System.out.println("Hello, world!"); JAVA语句括号内的字符串，重新编译即可输出


```
