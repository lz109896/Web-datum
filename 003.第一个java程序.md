```java
# Java代码
Java程序是大小写敏感的 对格式没有严格要求，但用空格或Tab键缩进会比较好看

# Hello.java

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, world!");
    }
}

## 文件名必须是Hello.java，文件名也要注意大小写，必须与程序的类名完全一致，扩展名是.java。

# 编译
切换到Hello.java所在目录：
javac Hello.java

## 无任何输出表示成功，可查看编译出的Hello.class文件。

# 执行
java Hello


```
