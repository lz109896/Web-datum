```java
try {
   // The guarded region:Dangerous activities
   // that might throw A,B, or C
} catch(A a1) {
   // Handler for situation A
} catch(B b1) {
   // Handler for situation B
} catch(C c1) {
   // Handler for situation C
} finally {
   //Activties that happen every time
}
```
#进入try的四种下场
##无疾而终
##发生了异常，而且捉到了
##发生了异常但是没有捉到
##发生了异常捉到了，但是又抛出了
##在离开try语句之前，最后都要进finally
