
#### 输出：PrintWriter
```
 format("格式",......)
 printf("格式",......)
 println(各种基本类型)
```
#### 输入：Scanner
```
 在 lnputStream 或 Reader 上建立一个 Scanner 对象可以从流中的文本中解析出以文本表达的各种基本类型
 next...()
```
#### Stream/Reader/Scanner
```
 数据是二进制的，用lnputStream
 数据不是二进制的，表达的是文本的，用Reader
 数据不是二进制的，表达的不是文本的，用Scanner

```
