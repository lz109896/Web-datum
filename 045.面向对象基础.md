#### class和instance是“模版”和“实例”的关系：

#### 类：class，对象模版
```
 class 定义了如何创建实例
 class 名字就是数据类型
```
#### 实例：instance，具体对象
```
 instance 是根据 class 创建的实例
 可以创建多个instance 
 各个 instance 类型相同，但各自属性可能不相同
```
#### 定义class
```
 一个class 可以包含多个field （字段）
 field 用来描述一个class的特征
 class 实现了数据封装
 通过变量、字段来访问实例字段

 class 是数据类型，instance 是数据
 class用字段（field）封装了数据，不同的instance拥有各自独立的field
 通过变量.字段名访问某个字段
 指向instance的变量都是引用变量

```
