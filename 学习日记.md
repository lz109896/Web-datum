### Javascript 语言基础
   今天复习的模块是 Javascript 语言基础，相对来说  Javascript 还是 比较容易理解；JavaScript一种属于网络的直译式脚本语言，已经被广泛用于Web应用开发，是一种动态类型、弱类型、基于原型的语言，内置支持类型。它的解释器被称为JavaScript引擎，为浏览器的一部分，广泛用于客户端的脚本语言，常用来为网页添加各式各样的动态功能,为用户提供更流畅美观的浏览效果。最早是在HTML（标准通用标记语言下的一个应用）网页上使用，用来给HTML网页增加动态功能。

1、是一种解释性脚本语言（代码不进行预编译）。
2、主要用来向HTML（标准通用标记语言下的一个应用）页面添加交互行为。  
3、可以直接嵌入HTML页面，但写成单独的 j s文件有利于结构和行为的分离。 
4、跨平台特性，在绝大多数浏览器的支持下，可以在多种平台下运行（如Windows、Linux、Mac、Android、iOS等）。

在1995年时，由Netscape公司的Brendan Eich，在网景导航者浏览器上首次设计实现而成。因为Netscape与Sun合作，Netscape管理层希望它外观看起来像Java，因此取名为JavaScript。但实际上它的语法风格与Self及Scheme较为接近。  
为了取得技术优势，微软推出了JScript，CEnvi推出ScriptEase，与JavaScript同样可在浏览器上运行。为了统一规格，因为JavaScript兼容于ECMA标准，因此也称为ECMAScript。

Javascript 脚本语言同其他语言一样，有它自身的基本数据类型，表达式和算术运算符及程序的基本程序框架。Javascript 提供了四种基本的数据类型和两种特殊数据类型用来处理数据和文字。而变量提供存放信息的地方，表达式则可以完成较复杂的信息处理。

### HTML&CSS
   HTML 加上 CSS 的组合，简直就是对前端页面给用户提供更多绚丽多彩的体验。在这么多天的学习复习中，HTML（Hypertext Markup Language）是文本标记语言，是用于描述网页文档的一种标记语言。CSS（Cascading Style Sheet）可译为“层叠样式表”或“级联样式表”，它定义如何显示 HTML 元素，用于控制Web页面的外观。通过使用CSS实现页面的内容与表现形式分离，极大提高了工作效率 。样式存储在样式表中，通常放在 <head> 部分或存储在外部 CSS 文件中。作为网页标准化设计的趋势，CSS取得了浏览器厂商的广泛支持，正越来越多的被应用到网页设计中去
   html是网页制作语言，css是布局的，以前没有css的时候，人们也都是用html来处理网页外观的，但html自身的处理属性与css相比就不行了，css的好处是容易管理，而且易于搜索引擎爬取，代码也相对减少，规范化。它们两一个书写内容，一个装扮，所以是一种完美的配合
   HTML是目前最流行的网页制作语言。互联网中的网页大多数都是使用HTML格式展示在浏览者面前的。随着Web 2.0概念的提出，要求网页具有更好的扩展性和用户体验，这使得CSS样式表在网页设计中变得越来越重要。为了能让广大的网页制作者快速地掌握网页制作的技巧和方法，第一部分以语法和实例相结合的形式详细讲解了HTML语言中各个元素及其属性的作用、语法和显示效果。第二部分从CSS基本概念开始，分别讲解了CSS盒模型和定位属性，CSS控制各种元素显示的方法，CSS布局页面的技巧等知识。第三部分为了增强读者的实战能力，以制作各种的形式详细演示了使用HTML和CSS制作页面的过程。为了便于理解，对所讲解的每个元素和属性，都做了实例演示。
   说简单其实也不简单，茫茫多的元素、属性、弄的晕头转向，主要还是需要记得懂得运用常用的元素和属性，其他的在项目实战中懂得去查找就行，关键点就是需要在脑海中有创建的模型，一点一点的去调试完善，“纸上得来终觉浅，绝知此事要躬行。”

### 智慧的提问 

##### How do I write a good title?我如何写一个好的标题？
   这里所说的标题并不是写文章的标题，但是有同样的属性，同样的道理。好的标题，是文章成功的一半。一个好的标题，首先是要抓住全文的核心、脉络，是对全文的概括。其次，好的标题应该富有吸引力。最后，标题应该具有一定的思想力度，能够透过标题告知读者想要的信息。标题写的好与坏，是吸引潜在读者的第一步。但不排除一些标题党，为了流量吸引读者，其实那些知识或者新闻早已不知被翻阅一百遍了。那么既然标题这么重要，那提问题的标题呢？同之，一个好的标题可以帮助你的问题得到应有的重视，标题的目的是吸引对你的问题感兴趣的人或者可以给你答案的人；人们一般看问题页面很快扫描一遍。那要让他们很容易注意到你问的问题？
   
   1.如果问问题比较多，建议分层次分步骤，尽量言简意赅，不要一大篇口水话，人家会没有多少兴趣阅读你的全部问题的
   
   2.表现问问题的主要中心，是某个领域的问题，最好提供所有的细节，所处环境，log日志，或者截图
   
   3.问问题当然少不了谦虚礼貌啦!!
   
##### How can I get answers fast?我如何快速获得答案？
   有时我在百度，看到一些问题我问的问题，很久了，仍然没有回答，但其他一样的问题，不同的标题得到了很快的回应。那在平时我们软件开发过程中怎么快速得到自己想要的答案呢？
   
1.在提问之前最好在专业级的浏览器 google 、百度找到答案，实在找不到，再找人帮助

2.尝试阅读手册、阅读常见问题文件（FAQ）、自己检查或调试，阅读源代码以找到答案。

3.自己在哪个行业工作，当然少不了加一些社区、社群、Q群、微信群，这些专业领域的交流工具，最起码预备不时之需

4.首先提问需要好标题，当你在群聊中被人看到，首先是需要能被人很清晰的读懂你的问题，人都是需要工作的，所以要勾起别人回答的欲望

5.尽量让问题容易阅读言简意赅，格式代码正确、行间缩进等

6.提供问题的背景：所处环境，log日志，或者截图

7.在问问题上使用正确的标签，也就是某个领域的问题，这样可以可以会更快捷

8.请不要随便在你的问题上标注“紧急”，因为是你的问题，不是别人的，别人没有义务帮你解决问题，看到紧急会让人感到反感
   
9.有句话说：使用 Google 的熟练程度看出一个程序员的水平，或许没有错。当我们首次输入问题是或许找不到满意的，这时就需要思考，问问题的是否出现错误，换换关键词试试？

### 一币一模特，一币一别墅
   标题听起来就是很牛逼哈，确实，这不是我们生活中的法币，暂时还不能在普遍用于生活中，技术上还是有挺多不完善的地方，确实出现过这么一种币：虚拟货币。但不同于我们在游戏中玩的游戏币、Q币。它依据特定算法，基于密码学的设计，通过大量的计算产生的可交易的货币。
   
   比特币（BitCoin）的概念最初由中本聪在2009年提出，根据中本聪的思路设计发布的开源软件以及建构其上的P2P网络。比特币是一种P2P形式的数字货币。点对点的传输意味着一个去中心化的支付系统。
   
   数货币不同，比特币不依靠特定货币机构发行，它依据特定算法，通过大量的计算产生，比特币经济使用整个P2P网络中众多节点构成的分布式数据库来确认并记录所有的交易行为，并使用密码学的设计来确保货币流通各个环节安全性。P2P的去中心化特性与算法本身可以确保无法通过大量制造比特币来人为操控币值。基于密码学的设计可以使比特币只能被真实的拥有者转移或支付。这同样确保了货币所有权与流通交易的匿名性。比特币与其他虚拟货币最大的不同，是其总数量非常有限，具有极强的稀缺性。该货币系统曾在4年内只有不超过1050万个，之后的总数量将被永久限制在2100万个。

   比特币可以用来兑现，可以兑换成大多数国家的货币。使用者可以用比特币购买一些虚拟物品，比如网络游戏当中的衣服、帽子、装备等，只要有人接受，也可以使用比特币购买现实生活当中的物品。 
   
   “只要有人接受”，其实意思就是大家都相信这个东西可以进行交易的币，形成的一种共识体系，很多人为什么会去买币，其实都是因为它有点像当年的股票，谁买谁赚钱的局势，经过这么多年的历史长河来看，K线也不是那么顺畅的，由于各个国家的管控，媒体的做空，也是起伏波荡的，分分钟破发都有可能，因为它没有上限下限，当初自己玩的时候亲眼看着首次上市，从10块钱的币值不到几天跌到几毛钱，这就是拿钱买刺激呀！！！
   
   虽然自己也炒过虚拟货币，但是还是比较单见浅闻，并没有深入了解过，以前在币圈流行的一句话就是一币一模特，一币一别墅，他们总想着一夜暴富，确实最近两年挺多人炒币一夜暴富的，还听过最奇葩的 ICO 买了几万的小蚁币，因为违法蹲局子3月出来后赚了两百多万，那是也是炒的最疯狂的时候，到处都是 ICO ，不排除很多都是空气币，传销币，也有人因此倾家荡产，背债度日的，妻离子散，跳楼的也不少。时下比特币从一币12万跌落至今日的41900，有人欢喜有人哭，看看自己以前买过关注过的币，貌似也都翻了几倍，但我已经是局外人。一同炒币的同事，以前他还没有我还炒波段，摸清楚规律，每天定时起床买卖，真的比什么积极。每天的币价波动就是自己心情的写照，这真的是得不偿失，哈哈，或许是自己内心不后强大。
   
   对于那些有极高信仰的人来说，那都是浮云，确实最近几年对于区块链的话题，越来越多，各大公司跑步入场，在马云的[一段演讲](http://www.sohu.com/a/233355358_323203)中提到，区块链只是一种去中心化得技术，未来10-20年以内，将有三大技术让人类面临挑战：机器智能、IoT和区块链。对于区块链，他认为区块链不是泡沫，但今天的比特币可能是泡沫。我认为整个区块链技术也正在发生天翻地覆的变化，区块链不是金矿，区块链是打开数字金融的一个金矿的巨大的工具和应用，是一个数据时代，隐私和安全的解决方案。
